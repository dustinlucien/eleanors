Excluding traffic by Cookie Content

This page will set a cookie in your browser that will help Google Analytics filter the traffic that comes from your computer when using this browser.

Once you have this setup, don't forget to set a filter in Google Analytics.

You will need to visit this page only once.

However, if you clear your browser cookies, reinstall the broser or operating system, make sure that you revisit this page, so the cookie will be set again.
